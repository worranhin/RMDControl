int RMD_GoAngleRelative ( int32_t angle)

Move the motor by a relative angle.

The function constructs a command packet to instruct the motor to move by the specified relative angle. The command is sent over an initialized serial communication channel.

参数
angleThe relative angle to move the motor, in 0.01 degrees.
返回
